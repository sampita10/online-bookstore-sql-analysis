# 📚 Online Book Store Data Analysis (SQL)

## 📌 Project Overview  
This project analyzes online book store data using SQL to find insights like sales performance, popular books, and customer ordering trends.

## 🛠 Tools Used  
MySQL  
SQL  

## 📂 Dataset  
This project uses datasets:
books.csv  
customers.csv  
orders.csv  

## 🔍 Analysis Performed  
Sales and revenue analysis  
Customer behavior analysis  
Book popularity analysis  
Category-wise performance analysis  

## 📊 Key Insights  
Some books generate higher revenue than others  
Certain customers contribute to majority of sales  
Different categories perform differently  
Orders vary based on time and demand trends
